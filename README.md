🛠 Getting Started
Prerequisites

Node.js (v18+)

npm or yarn

Installation
# Clone the repository
git clone https://github.com/shivasai-muthyam/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev


Visit http://localhost:5173 in your browser.
